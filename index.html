Axioma

----------------

Plug-ins de áudio de sintetizador modular desenvolvidos para a biblioteca VCV Rack de código aberto. Os plugins foram baseados em modelos matemáticos específicos de comportamento caótico e topologia, onde a representação visual de tais modelos é usada como método de controle ou síntese (código-fonte completo). A lista atual de plugins Axioma é a seguinte:

The Bifurcator: geração de sequência de controle de acordo com as posições mapeadas no mapa de bifurcação. As posições são obtidas com base em iterações de um tipo de função selecionado, que de acordo com o valor de um parâmetro de bifurcação, pode levar a um comportamento caótico;

Tesseract: controle CV de saída com base na posição de cada vértice de um Tesseract giratório projetado em um espaço 2D;

Ikeda: geração de sequência CV baseada no mapa de sistema dinâmico proposto por Kensuke Ikeda, que descreve um comportamento caótico dentro de uma faixa de parâmetros específica.

Rhodonea: VCO com dois osciladores definidos pelas curvas Rhodonea. Pode-se considerar o método utilizado como síntese poligonal, onde ondas são criadas a partir de uma projeção de coordenadas polares em coordenadas cartesianas.

SuperCollider

Scripts

----------------

SuperCollider é uma plataforma de síntese/processamento de áudio que utilizo em performances sonoras, instalações, composições e também como ferramenta de protótipo de desenvolvimento de software de áudio. Aqui estão alguns exemplos de scripts:

-> ODOR MOLECULE SONIFICATION: Script de um programa com GUI que converte medições do espectro infra-vermelho de moléculas de odor em componentes de síntese aditiva e outros recursos para composição de sequência sonora usando diferentes moléculas;

-> BIRDS AGAINST THE MACHINE: Script utilizado em performance para criar uma batalha sonora entre pássaros e motosserras.

-> SYNTHESIS OF LIFE: Script utilizado em performance de síntese sonora através do jogo da vida (para ser usado em combinação com Korg nanoKONTROL 2 como controlador MIDI).

-> POLAR WAVE SHAPER: Script com GUI para criação de formas de onda (método wavetable) através de desenhos de formas em coordenadas polares.

-> WORKSHOPS sobre SuperCollider: workshops que venho criando para compartilhar o conhecimento sobre essa ferramenta incrível!

Web apps

----------------

A lista abaixo mostra alguns aplicativos de código aberto que criei para serem usados ​​como ferramentas para design acústico (foram desenvolvidos com um foco para uso em navegadores de desktop):

Parabolic microphone gain: Ferramenta gráfica para estimativa da amplificação acústica do microfone quando conectado a uma antena parabólica. A resposta estimada é para uma onda plana incidente com angulo de 0 graus. Pode-se definir as dimensões para projetar o formato parabólico para uma amplificação esperada;

Coupled acoustic resonators: Ferramenta gráfica para estimativa da resposta acústica no projeto de ressonadores acústicos (ressonadores de Helmholtz). É possível obter valores de impedância e absorção ao longo da faixa de frequência audível, com base nas dimensões específicas solicitadas e nas propriedades do material de absorção. Também é possível usar a ferramenta para projetar ressonadores acopladas (até 3 ressonadores);

Waves on canvas: Ferramenta gráfica para visualização de campos acústicos criados pela combinação de diferentes tipos de fontes pontuais (monopolo, dipolo, quadrupolo). Pode ser usada como ferramenta de ensino para mostrar de forma interativa como diferentes fontes desempenham um papel no campo acústico em relação à amplitude, fase, frequência e direção.

Áudio

----------------

Após meu mestrado, passei 3 anos trabalhando com projetos de pesquisa em processamento de sinais de áudio relacionados ao desenvolvimento de modelos do nervos auditivo humano, tanto no LVA-UFSC quanto no AIP-TUM (onde também trabalhei como assistente de ensino no curso de processamento de sinal para tecnologia de áudio) .

Nos últimos anos (2021 - 2024), trabalhei como desenvolvedor em processmento digital de sinais de áudio para SoCs (System on a Chip) em Absolute Audio Labs, onde fui responsável pelo design de software, validação e ajuste de aplicativos de fala e música com foco principal em pessoas com perda auditiva (firmware para fones de ouvido e aparelhos auditivos).
